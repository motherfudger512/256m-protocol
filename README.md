 # 256M Insurance Protocol

 Decentralized mutual insurance protocol on Solana for real-world assets.

 ## Overview

 256M is a modular insurance protocol enabling on-chain insurance policies with:
 - Decentralized coverage management
 - LP capital pools with yields
 - AI-powered fraud detection
 - Swiss regulatory compliance

 ## Quick Start

 ```bash
 # Install
 yarn install

 # Build
 anchor build

 # Deploy to devnet
 ./deploy-devnet.sh

 # Initialize
 yarn initialize-devnet

 # Test
 yarn test-journey
 ```

 ## Architecture

 - **Liquidity Pool**: Capital management, LP deposits/withdrawals
 - **Policy Manager**: Customer KYC, policy lifecycle
 - **Claims Processor**: Claims review & payouts
 - **Treasury**: Fee management
